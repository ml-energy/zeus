    run

    zeus.run

    Classes for running actual jobs with Zeus.

    ZeusDataLoader is to be embedded in the training script, automatically taking care of profiling power and determining the optimal power limit. ZeusMaster is the top-level class that drives batch size optimization (with Multi-Armed Bandit) and spawns training jobs.
